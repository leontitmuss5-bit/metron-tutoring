# Métron Tutoring App

This is a Vite + React application for Métron Tutoring.

## Prerequisites

You need **Node.js** installed on your system to run this project.
Download and install it from [nodejs.org](https://nodejs.org/).

## Getting Started

1.  Open a terminal in this directory:
    ```bash
    cd metron-app
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open your browser at `http://localhost:5173` (or the URL shown in the terminal).

## Project Structure

-   `src/components`: Shared components like Navbar and Footer.
-   `src/pages`: Page components (Home, Tutors, Pricing, TutorProfile).
-   `src/index.css`: Global styles and Tailwind directives.
-   `tailwind.config.js`: Tailwind CSS configuration with custom theme colors.
